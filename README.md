## 1. วัตถุประสงค์ของโปรแกรม
โปรแกรมนี้ถูกพัฒนาขึ้นเพื่อช่วยในการจัดการสูตรอาหาร โดยมีฟังก์ชันหลักในการเพิ่ม แก้ไข ลบ และค้นหาสูตรอาหารได้อย่างสะดวกและเป็นระบบ ผู้ใช้สามารถบันทึกส่วนผสมและขั้นตอนการทำอาหารลงในฐานข้อมูลเพื่อให้สามารถเรียกดูและจัดการสูตรอาหารได้อย่างง่ายดาย

## 2. พื้นฐานและเหตุผล
ปัจจุบันผู้ที่รักการทำอาหารมักจะจดบันทึกสูตรอาหารของตนเองไว้ในกระดาษหรือไฟล์เอกสาร ซึ่งอาจทำให้เกิดความยุ่งยากเมื่อต้องการค้นหาหรือจัดการสูตรอาหาร โปรแกรมนี้จึงถูกพัฒนาเพื่อแก้ปัญหาดังกล่าว โดยใช้ฐานข้อมูล SQLite ในการจัดเก็บข้อมูล และใช้ Tkinter สำหรับสร้างอินเทอร์เฟซที่ใช้งานง่าย เพื่อให้ผู้ใช้สามารถเข้าถึงและบริหารจัดการสูตรอาหารของตนได้อย่างมีประสิทธิภาพ

## 3. ขอบเขตการทำงาน
- รองรับการเพิ่มสูตรอาหารใหม่ลงในระบบ พร้อมบันทึกชื่อ ส่วนผสม และขั้นตอนการทำอาหาร
- รองรับการแก้ไขและลบสูตรอาหารที่มีอยู่
- สามารถค้นหาสูตรอาหารจากชื่อได้เพื่อความสะดวกในการใช้งาน
- แสดงรายการสูตรอาหารทั้งหมดในรูปแบบตารางเพื่อให้ง่ายต่อการเลือกและจัดการ
- ใช้ฐานข้อมูล SQLite ในการจัดเก็บข้อมูล เพื่อให้สามารถบันทึกข้อมูลแบบถาวร
- รองรับการใช้งานบนระบบปฏิบัติการ Windows ผ่านอินเทอร์เฟซที่สร้างด้วย Tkinter

## 4. ประโยชน์ของโครงการ
- ช่วยให้ผู้ใช้สามารถจัดเก็บและบริหารจัดการสูตรอาหารได้อย่างเป็นระบบ
- ลดปัญหาการสูญหายของสูตรอาหารที่บันทึกไว้ในรูปแบบกระดาษหรือไฟล์ที่ไม่เป็นระเบียบ
- เพิ่มความสะดวกในการค้นหาและเรียกดูสูตรอาหาร ทำให้ประหยัดเวลา
- รองรับการใช้งานง่ายด้วยอินเทอร์เฟซที่เป็นมิตรต่อผู้ใช้
- สามารถพัฒนาและขยายฟังก์ชันเพิ่มเติมได้ในอนาคต เช่น การแบ่งปันสูตรอาหารหรือเชื่อมต่อกับแอปพลิเคชันอื่น ๆ
